# csharp-modern-controls

엔터프라이즈 Windows 데스크톱용 모던 WPF 컨트롤 라이브러리.
새 UI는 순수 WPF로 작성하고, 기존 WinForms 프로그램에는 `ElementHost` 래퍼로 호스팅합니다.

- .NET Framework 4.8 · C# 7.3 · Visual Studio 2026

- Claude Code 작업 규칙: [`CLAUDE.md`](CLAUDE.md)
